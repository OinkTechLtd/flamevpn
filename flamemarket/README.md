# 🔥 FlameMarket - Кайфуйте от интернета вместе с FlameMarket

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
[![React](https://img.shields.io/badge/React-19.2.6-blue.svg)](https://reactjs.org/)
[![Vite](https://img.shields.io/badge/Vite-8.0.14-purple.svg)](https://vitejs.dev/)

**FlameMarket** — это современный магазин приложений нового поколения по аналогии с HappyMod, Google Play Store, RuStore и App Store. 

Каталог включает модифицированные приложения (моды) и оригинальный софт. Скачивайте безопасно и кайфуйте!

![FlameMarket Preview](preview.png)

## ✨ Особенности

### 🔥 Основные возможности
- **🎮 Моды и Оригинальный Софт** — огромный каталог модифицированных приложений и оригинального ПО
- **🛡️ Безопасность** — проверка через VirusTotal, защита от скам-приложений
- **🔒 VPN Поддержка** — работает с VPN для доступа без ограничений
- **👶 Детский Режим** — безопасное использование детьми
- **⚡ Быстрые Загрузки** — прямые ссылки на сайты разработчиков
- **🤖 Авто-обновления** — поисковый робот добавляет новые приложения

### 🎨 Современный Дизайн
- В стиле RuStore / App Store / Google Play Store
- Адаптивный интерфейс для мобильных и ПК
- Темная тема с огненными акцентами
- Плавные анимации и переходы

## 🚀 Быстрый старт

### Требования
- Node.js 18+ 
- npm или yarn

### Установка

```bash
# Клонируйте репозиторий
git clone https://github.com/yourusername/flamemarket.git
cd flamemarket

# Установите зависимости
npm install

# Запустите в режиме разработки
npm run dev
```

Приложение откроется по адресу `http://localhost:3000`

### Сборка для продакшена

```bash
# Создайте production сборку
npm run build

# Предварительный просмотр
npm run preview
```

## ⚙️ Настройки безопасности

В приложении доступны следующие настройки:

| Функция | Описание |
|---------|----------|
| 👶 Детский режим | Блокирует 18+ контент и опасные приложения |
| 🔞 Блокировка 18+ | Скрывает приложения с возрастным ограничением |
| 🛡️ Защита от скама | Блокирует приложения с низким рейтингом безопасности |
| 🔒 VPN поддержка | Разрешает работу через VPN |

## 📊 Система безопасности

FlameMarket использует многоуровневую систему проверки:

1. **VirusTotal Integration** — автоматическая проверка на вирусы
2. **Рейтинг безопасности** — отображается для каждого приложения (0-100%)
3. **Предупреждения** — уведомления о потенциально опасных приложениях
4. **Поисковый робот** — автоматически сканирует и добавляет новые приложения

### FAQ

#### ❓ Что такое FlameMarket?
FlameMarket — магазин приложений с модами и оригинальным софтом.

#### ❓ Безопасно ли использовать моды?
Мы проверяем все приложения через VirusTotal. Рекомендуем иметь антивирус.

#### ❓ Как включить детский режим?
Зайдите в настройки и включите опцию "Детский режим".

#### ❓ Работает ли с VPN?
Да, полностью поддерживает работу с VPN.

#### ❓ Как скачать приложение?
Найдите приложение в каталоге, нажмите "Скачать" — откроется сайт разработчика.
